## 🚀 About Project

🐾 **FurEver Care** is a modern and responsive pet care website designed to provide a complete solution for pet lovers.

It includes:
- 🐶 Pet Products  
- 🏥 Veterinarian Services  
- 🚨 Emergency Support  
- 🏠 Pet Adoption & Shelters  

This project focuses on **UI/UX design, animations, and responsive layouts** to deliver a real-world experience.

---

## ✨ Features

- 🎨 Clean & Modern UI  
- 📱 Fully Responsive Design  
- 🎬 Smooth Animations (AOS + CSS)  
- 🐾 Interactive Effects  
- 📩 Newsletter Subscription (Frontend)  
- 🔝 Scroll & Hover Effects  

---

## 🛠️ Tech Stack

- 💻 HTML5  
- 🎨 CSS3  
- ⚡ JavaScript  
- 🎯 Bootstrap 5  
- 🎬 AOS Animation Library
